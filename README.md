#### 双指针
两个指针解决一个问题
- 普通双指针: 两个指针往同一个方向
- 对撞双指针: 两个指针面对面移动 
    - 一般是有序的
    - 两个指针一个头一个尾
- 快慢双指针: 慢指针 + 快指针
    - 一般是环形链表找重复
    - 指针一个走一步, 一个走两步
